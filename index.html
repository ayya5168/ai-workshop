Antigravity 一鍵設定

安裝 superpowers + GitHub MCP + Railway MCP

偵測到你的系統: …
1在 Antigravity 內建 terminal 貼上這行
2按 Enter 等它跑完
大約 30 秒。期間會自動安裝 Superpowers 並寫入 MCP 設定檔。
3照 terminal 的指示完成 3 個手動步驟
  1. 到 GitHub Settings → Tokens 建 PAT，scopes 勾 repo, read:org, read:user。編輯 ，把 YOUR_GITHUB_PAT 換成 token
  2. 裝 Railway CLI 並登入: → railway login
  3. 完全關掉 Antigravity 再打開（不是 reload window）
常見問題
  • terminal 找不到 gemini → 請確認從 Antigravity 內建 terminal 開（不是系統 terminal）
  • npm 裝 railway 報 EACCES → macOS 改用 brew install railway；Windows 用 winget install Railway.Railway
  • PAT scope 不確定 → repo, read:org, read:user 三個就夠
  • 裝完看不到 server → 一定要「整個 quit」Antigravity，reload window 不會重讀 MCP 設定
